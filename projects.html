Projects

Project 1: Java Epoch Time Converter

This program takes 3 separate inputs from the user: a month (in the form of a digit from 1-12); a day (in the form of a digit from 1-31); and a year (any valid year). It then computes that date in epoch time, and prints this value to the screen. Finally, the program also selects a random date from the same year that the user entered, and also prints this value to the screen. The program takes leap years into consideration to ensure that all printed values are correct. While this project is a little simple compared to what I could code now, I chose to exhibit it on my site because it was really the first major project I did after declaring my Computer Science major, and it feels special to me because of that.
